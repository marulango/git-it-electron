Social Coding

Trabajar con otras personas es una de las mejores cualidades de GitHub porque hace tán facil el trabajar en cualquier parte del mundo a cualquier hora.

Collaborators are other GitHub users who are given permission to make edits to a repository owned by someone else. Contributors are other GitHub users who have made contributions to a project.

@reporobot is a robot from outer space that loves collaborating on repos.

Hola, Repo Robot!

Para agregar un colaborador a uno de tus proyectos, visita la página de tu repositorio en tu cuenta de GitHub y selecciona el tab de 'Settings' en el menú de la parte de arriba de la página. Después selecciona la opción de 'Collaborators'. Escribe el nombre de usuario en la caja de texto y selecciona 'Add'.

Go to your forked Patchwork repository's page on GitHub and add 'reporobot' as a collaborator. The URL should look like this, but with your username.

github.com/YOURUSERNAME/patchwork/settings/collaboration
