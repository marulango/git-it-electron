Git

Git es un software gratuito de código abierto (la fuente del código es pública) escrita por Linus Torvals quien también escribió el sistema operativo de Linux.

Git es un programa para hacer un seguimiento de cambios a través del tiempo, conocido en programación como version control. Si tú has usado un seguimiento de cambios en un documento de texto ¡entonces ya estás familiarizado con el concepto!

Instala Git

Nosotros recomendamos instalar Git en tu computadora descargando la aplicación de GitHub Desktop. No usaremos la aplicación de escritorio en Git-it (¡estamos aprendiendo de terminales!) pero eso incluye Git y es la manera más fácil de instalar Git en todas las plataformas de la misma manera.

  • Windows: Usa la Git Shell para tu terminal.
  • Mac: Abre GitHub de escritorio y ve a Preferencias, selecciona la línea de comandos con herramientas para instalar.Usa la aplicación terminal como tu termianal.

¿Ya tienes Git o no estás seguro? Escribe git --version y si eso regresa a una versión numérica arriba de 1.7.10,¡Estás listo! Para más información, visita Git website.

Git no es como otros programas en tu computadora. No verás el logo en tu escritorio, pero siente estará disponible para ti y podrás acceder cada vez que quieras desde tu terminal o aplicaciones de escritorio de Git.

Configura Git

Una vez que Git está instalado, abre tu terminal. Puedes verificar que está realmente ahi escribiendo:

git --version

Esto regresará a la versión de Git que estás ejecutando y se ve algo así:

git version 1.9.1

Después, configura Git y esto sabe asociar tu trabajo para ti:

Establece tu nombre:

git config --global user.name "Your Name"

Ahora establece tu correo:

git config --global user.email "youremail@example.com"

Signos de Dollar en Documentación de Código

Los signos de dollar $ son usados a menudo en programación de documentación para indicar que la lines es un código command line (Ve el fragmento del código abajo). No tienes que escribir el $, solo escribe lo que viene después.
