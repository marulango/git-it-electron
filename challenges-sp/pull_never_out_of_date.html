Atrae de un remoto

Si estás trabajando en algo con alguien más necesitas estar al día de los últimos cambios. Querrás atraer cualquier cambio que pudo hacer sido dejado en la central de repositorios de GitHub.

¿En qué ha estado Reporobot?

Revisa si Reporobot ha hecho algún cambio en tu rama atrayendo por el remoto llamado ‘origen’ en GitHub:

git pull <REMOTENAME> <BRANCHNAME>

Si nada ha cambiado, este te dirá ‘Al día’. Si hay algún cambio, Git fusionara esos cambios a tu versión local.

¿Reporobot hizo cambios? Git te dirá donde se hicieron los cambios. Puedes abrir ese archivo y ver las actualizaciones de Reporobot. Sorpresa, ¡Reporobot es un artista!

Favor de seleccionar su directorio de repositorio
  • Revisar el estado de Git
    • git status
  • Atrae cambios a tu rama remota
    • git pull <REMOTENAME> <REMOTEBRANCH>
  • Ver cambios del remoto antes de atraer
    • git fetch --dry-run
