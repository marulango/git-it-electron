Trabajando mejor, juntos

El repositorio que habías creado hasta el momento se encuentra en tu computadora, lo cual es práctico, pero hace muy difícil compartir y trabajar con otros. No te preocupes, ¡para eso es GitHub!

GitHub es una página web que permite a las personas de todos lugares subir en lo que están trabajando con Git, así todos pueden trabajar juntos fácilmente en el mismo proyecto. En vez de tener tu proyecto solo en tu computadora, también puedes sincronizarlo en GiHub el cual actúa como un repositorio central para ti y los demás para compartir.

Crea una Cuenta de Github

Visita github.com y regístrate para una cuenta nueva. ¡Dame esos cinco, bienvenido!

Agrega tu nombre de usuario de GitHub a Git

Agrega tu nombre de usuario de GitHub a tu configuración Git. Haremos esto solo por el bien de Git-it, esto hace más fácil verificar los próximos desafíos. Guárdalo exactamente como cuando lo creaste en GitHub — capitalizar en mayúscula, y recuerda, no necesitas dar espacios en "<" and ">".

Agrega tu nombre de usuario de GitHub en tu configuración:

git config --global user.username <USerNamE>

Puedes volver a comprobar lo que acabas de hacer en tu configuración de Git escribiendo: git config --global user.username en la terminal.

Asegúrate de escribir "user.username" y no "user.name", el cual puede anular el nombre que asignaste en el primer desafío y ¡dejarte sin nombre de usuario propio! Si te diste cuenta que hiciste eso, está bien, solo repite el paso del primer desafío y agrega tu nombre y después completa el paso de arriba.

Configuración de nombres de usuario de GitHub & Git que no coinciden

Un error común es no tener tu nombre de usuario de Github coincidiendo en el caso del cual configuraste con git config. Por ejemplo, 'JLord' no es lo mismo que 'jlord'

To change your username set with Git, just do the same command you did earlier, but with the correct capitalization:

git config --global user.username <USerNamE>

Cuando ya hayas hecho tus actualizaciones, ¡Verifica de nuevo!
