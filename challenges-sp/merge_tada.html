Limpieza

Una vez que no necesites las ramas nunca más puedes eliminarlas local y remotamente. Esto es útil para que no termine una pila de ramas viejas.

Fusiona localmente

¡Tú solicitud de atraer fue fusionada! Tu rama fue fusionada en la rama original de GitHub de “gh-pages”. Puedes fusionar tu rama localmente, también.

Primero, muévete en la rama que quieres fusionar---- en este caso, la rama ‘gh-pages’

git checkout gh-pages

Ahora dile a Git que rama vas a fusionar---- en este caso, tu rama característica que empieza con “add-“

git merge <BRANCHNAME>

Poner en orden mediante la supresión de la rama característica. Ahora esta fue fusionada, en realidad no la necesitas alrededor.

git branch -d <BRANCHNAME>

También puedes eliminar la rama de tu fork en Github:

git push <REMOTENAME> --delete <BRANCHNAME>

Atraela a Upstream

Y lo último pero no menos importante, el original cambió desde que tu solicitud de atraer fue fusionada--- ¡Reporobot agregó tu nombre a la página web! Si tú empujas en estos cambios para el upstream, podrás estar al día y tener esa versión también.

Para atraer desde el upstream original:

git pull upstream gh-pages
